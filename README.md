# notion-embeds

Static HTML files published via GitHub Pages for Notion embed use.

## Usage

1. Place HTML files in repo root.
2. Push to `main`.
3. Access via `https://yujiro-a.github.io/notion-embeds/<filename>.html`.
4. In Notion: `/embed` → paste URL.

## Files

- `profile_VicePresident.html` — Yujiro Arakawa self-introduction (Vice President profile).
